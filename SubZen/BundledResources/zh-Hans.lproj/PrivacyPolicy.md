# 隐私政策

## 概述

SubZen 是一款隐私优先的订阅管理工具,所有数据保留在您的设备上。本政策说明我们对您隐私和数据的处理方式。

## SubZen 团队隐私政策

- **无数据收集**: 我们不收集、存储或传输任何个人信息或使用数据。
- **所有数据保留在本地**: 您的订阅数据完全保留在设备上,存储在 iOS 沙盒存储 (UserDefaults) 中。
- **无遥测数据**: 我们不跟踪使用情况、行为模式或分析数据。
- **无崩溃报告**: 我们不收集崩溃报告或诊断数据。
- **无需账号**: SubZen 完全离线工作,无需注册、无服务器、无云端。

## 网络使用

SubZen 仅在以下情况发起网络请求:
- **货币汇率**: 从 [Frankfurter API](https://frankfurter.app) (欧洲中央银行数据) 获取实时汇率,用于跨货币转换订阅费用。这些请求不包含任何个人信息。
- **离线回退**: 离线时自动使用缓存汇率 (最长 24 小时)。

如果您的所有订阅使用同一货币,可以完全离线使用 SubZen。

## 第三方服务

SubZen 不集成任何第三方分析、广告或追踪服务。

## 数据安全

- 您的数据受 iOS 沙盒和设备加密保护 (启用密码/面容 ID 时)。
- 我们不向任何服务器传输订阅详情。
- 货币 API 请求仅发送货币代码 (如 "USD"、"CNY"),不包含您的订阅数据。

## 您的权利

由于所有数据存储在本地:
- **导出**: 使用 iOS 备份在切换设备时保留数据。
- **删除**: 删除应用即可永久移除所有存储数据。
- **重置**: 在设置中使用 "重置所有数据" 清除订阅而不卸载应用。

## 政策更新

我们可能在添加新功能时更新此政策,更新内容将在应用和 GitHub 仓库中发布。
